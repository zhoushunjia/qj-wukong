核心记忆点：“初”

关键词：阴阳剪、剪刀、简约、时尚

“初”— —意为“回归最初的本质”，本质意为本源，本源即为“道”，是中国 古代哲学的重要范畴，老子把道看做是宇宙的本原和普遍规律，回归最初的本质 即回归道，所谓生一，一生二，二生三，三生万物，“道生一，一生二”，意思 是指，在谁也不知道的地产生了本源之气，这个本源又分化阴与阳这两种气，这 就是最初代表这一观点最完美的图形即是阴阳图，初简的标志即来与阴阳图表现 出核心记忆点“初”.

剪刀— —剪刀是剪头发的必备工具，本设计“初”与剪刀结合、拆分、重组，将 寓意与实用结合，让传统与现代碰撞出新的火花.

颜色用黑色单色表现，提现时尚、简约、男性为主.
