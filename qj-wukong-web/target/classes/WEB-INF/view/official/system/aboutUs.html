公司简介

我们处在一个信息革命的时代，这样的一个时代，造就了Microsoft、Oracle等伟大的基础软件企业。 可以预见，在这样一个时代，也必将造就伟大的应用软件企业。我们处在一个中国经济迅速崛起的时代，这样的一个时代，在中国大地上， 造就了华为、阿里巴巴等伟大的企业。可以预见，随着中国经济的崛起，在中国大地上将造就更多伟大的IT企业。 我们坚守“客户第一”的经营理念，深刻理解客户需求，持续提高技术和服务水平；我们坚持“以人为本”的企业文化， 尊重知识、珍爱人才；我们倡导“开放、平等、协作、分享”的互联网精神，向客户学习，向同行学习；我们不断变革创新、锐意进取，争创一流！
